# Chat Agents

This directory contains chat-specific agent implementations.

## Purpose

Chat agents handle:
- Natural language conversation
- Intent recognition
- Response generation
- Context management
- Integration with chat skills

## Structure

Place your chat agent files here, such as:
- `agent_config.yaml` - Agent configuration
- `agent.py` - Agent implementation
- `prompts/` - Agent prompts and templates
- `context/` - Context management modules
