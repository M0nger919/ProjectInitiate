# Skills

This directory contains reusable skills that can be used by agents.

## Structure

- **chat/**: Chat-related skills and utilities

## Purpose

Skills are modular capabilities that:
- Perform specific tasks
- Can be composed together
- Are reusable across different agents
- Provide well-defined interfaces

## Usage

Each skill subdirectory should contain:
- Skill implementation code
- Skill configuration
- Skill documentation
- Unit tests
