# Chat Skills

This directory contains chat-specific skills.

## Purpose

Chat skills provide capabilities such as:
- Message parsing and formatting
- Sentiment analysis
- Language translation
- Entity extraction
- Response templating
- Context retrieval

## Structure

Place your chat skill files here, such as:
- `message_parser.py` - Parse incoming messages
- `response_formatter.py` - Format outgoing responses
- `context_manager.py` - Manage conversation context
- `skill_config.yaml` - Skill configurations
