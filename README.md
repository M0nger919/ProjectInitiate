# ProjectInitiate

Document repo for starting project, ideas, agent.

## Structure

### Agents
The `agents/` directory contains agent implementations:
- **agents/chat/**: Chat-specific agents and configurations

### Skills
The `skills/` directory contains reusable skills:
- **skills/chat/**: Chat-specific skills and utilities

## Getting Started

1. Navigate to `agents/chat/` to create or configure chat agents
2. Navigate to `skills/chat/` to create or configure chat skills
3. Refer to README files in each directory for detailed documentation

## Example Files

Example configuration files are provided in each directory to demonstrate the structure:
- `agents/chat/example_agent_config.yaml`
- `skills/chat/example_skill_config.yaml`
